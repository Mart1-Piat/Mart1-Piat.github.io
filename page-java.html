Projet Java & Java Swing

Projet IUT de développement logiciel réalisé en groupe de quatres, visant à simuler la gestion administrative de la Cité Internationale Universitaire de Paris. Ce projet combinait plusieurs compétences clés acquises en cours d’IHM, de développement orienté objet et de qualité de code.

L’objectif était de concevoir une application permettant de gérer des entités comme les maisons étudiantes, les services proposés, ainsi que l’inscription et la répartition des étudiants en fonction de critères comme la nationalité ou les disponibilités. Nous avons modélisé l’ensemble des objets via un diagramme UML puis implémenté la logique métier en Java, avec une structure orientée objet rigoureuse. Les classes ont été encapsulées et générées depuis Visual Paradigm avant d’être enrichies manuellement.

Nous avons également conçu une interface utilisateur graphique avec Java Swing, en nous appuyant sur des maquettes et storyboards produits via Figma. Cette interface permettait d’interagir de manière fluide avec les fonctionnalités principales de l’application.

Enfin, une part importante du projet portait sur la qualité logicielle : maintien d’un dépôt Git collaboratif, mis en place des tests unitaires en amont de l’implémentation et auto-évaluation du code final. Ces pratiques nous ont permis d’acquérir une méthode de développement rigoureuse et collaborative, essentielle dans un contexte professionnel.
